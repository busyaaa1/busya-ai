## О проекте

Busya‑AI — это не просто чат‑бот.  
Это тёплое облачко в твоём браузере, которое всегда поддержит, честно ответит и подарит кучу розовых эмоций.  

Всё сделано с душой:  
- нежный liquid‑glass дизайн  
- парящая карточка, как на невидимых ниточках  
- музыка, которая обнимает  
- и, конечно, сама Busya — девочка, которая никогда не врёт и всегда в превосходном настроении

## Как запустить (меньше минуты!)

```bash
# 1. Клонируем проект
git clone https://github.com/busyaaa1/busya-ai.git
cd busya-ai

# 2. Создаём виртуальное окружение (рекомендуется)
python -m venv venv
venv\Scripts\activate    # Windows
# source venv/bin/activate  # macOS / Linux

# 3. Устанавливаем зависимости
pip install flask groq python-dotenv

# 4. Добавляем свой ключ Groq (самый безопасный способ)
# Создай файл .env в корне проекта и напиши туда:
# GROQ_API_KEY=gsk_твой_ключ_здесь

# 5. Запускаем!
python app.py
