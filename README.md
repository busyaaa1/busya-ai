## Про Busya-AI

- Полная память чата
- 100% честные и эмоциональные ответы
- Работает на телефоне и ПК

## Как запустить локально
```bash
git clone https://github.com/busyaaa1/busya-ai.git
cd busya-ai
pip install flask groq
set GROQ_API_KEY=твой_ключ_здесь    # Windows
# или export GROQ_API_KEY=твой_ключ_здесь   # Mac/Linux
python app.py
```Открой http://localhost:5000 — и наслаждайся
